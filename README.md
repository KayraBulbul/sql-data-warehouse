# Data Warehouse and Analytics Project

This project is a data warehousing and analytics solution, from building a data warehouse to generating actionable insights.

## Objective

My objective with this project is to develop data warehouse using SQL Server and using SQL-based analytics, to deliver detailed insights into:
- **Customer Bahavior**
- **Product Performance**
- **Sales Trends**
